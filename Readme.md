# C2Gether

Watch videos on sync with every member of the room, with video controls access

> Live link - [C2Gether](https://collab-study-client.vercel.app/)

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/HLrahul/C2Gether.git
    ```

2. Navigate to the client directory and install dependencies:

    ```sh
    cd client
    npm install
    ```

3. Navigate to the server directory and install dependencies:

    ```sh
    cd ../server
    npm install
    ```

### Running the Application

1. Start the client:

    ```sh
    cd client
    npm run dev
    ```

2. Start the server:

    ```sh
    cd ../server
    npm run dev
    ```

### Building the Application

1. Build the client:

    ```sh
    cd client
    npm run build
    ```

2. Build the server:

    ```sh
    cd ../server
    npm run build
    ```

### Running the build

1. Run client and server with same command in both terminals

   ```sh
   npm run start
   ```
